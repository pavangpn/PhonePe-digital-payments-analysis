## PhonePe-digital-payments-analysis
    Analyzing transaction data from the financial application PhonePe along with demographic data from various states and districts in 
    India.

## 🔹 Introduction
>>   Digital payments have become a cornerstone of India’s financial ecosystem, with platforms like PhonePe driving rapid adoption across        states and districts.
>>   This project explores PhonePe transaction data to uncover meaningful insights into usage trends, demographic influences, and
     behavioral patterns.

## 🎯 Objective
     To transform analyze the transaction data from the financial application PhonePe along with demographic data from various states and 
     districts in India to uncover meaning insights.

## 🛠️ Tools & Libraries
    Python
    Pandas
    Numpy
    Matplotlib
    Seaborn
    Jupyter Notebook
